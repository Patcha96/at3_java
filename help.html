User Guide

Legend:

  1. The options dropdown menu allows you to select between the User Guide and the JavaDoc
  2. The window controlls allow you to minimize, maximize and close the program.
  3. Name text field where you input any item you would like to be hashed and added to the list.
  4. Hash text field where the hashed value of selected item shows up.
  5. Add Button allows you to add the item you put in the name text field into the list.
  6. Search Button allows you to search the list for items that match the name you put into the name text field.
  7. Remove Button allows you to remove any item from the list that matches the name in the name text field.
  8. List of items already added to the list with the aforementioned controls, you can select any item from the list and it will display its hashed value in the hash text field.
